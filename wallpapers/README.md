# wallpapers

Place your default wallpaper in this directory as:

```text
default.png
```

Recommended workflow:
- add `wallpapers/default.png` before running `install.sh`, or
- after install, copy your wallpaper to `~/Pictures/Wallpapers/hyprglass/default.png` and run:

```bash
~/.local/bin/theme-apply ~/Pictures/Wallpapers/hyprglass/default.png
```

Additional wallpapers can live in the same directory. `theme-random` will pick from the installed wallpaper directory on the target system.
